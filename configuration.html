Authentication Setup *
Hybrid Model: Addocu requires both OAuth2 (automatic) for permissions AND your Google Cloud API Key (manual) for API access. Both are mandatory.
How to get your API Key:
1. Go to Google Cloud Console
2. Create or select a project
3. Enable APIs: Analytics Admin, Tag Manager, Looker Studio
4. Create API Key in "Credentials" section
5. Copy and paste it above
How to get your Developer Token:
1. Go to Google Ads API Center
2. Apply for API access (if not already approved)
3. Copy the Developer Token from "API access details"
4. Paste it above to enable Google Ads audits
